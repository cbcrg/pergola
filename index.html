Welcome to Pergola’s documentation!¶

Pergola is a python library that provides a framework for allowing to explore, integrate and analyze your behavioral data using genomic tools. in common genome browser

We here present the Python bEhavioRal GenOmebrowser LibrAry (Pergola), a python library that makes longitudinal behavioral data compatible with popular desktop genome browsers. Pergola accomplishes data format conversion using a simple correspondence between behavioral and genomic data. Behavioral longitudinal data can then be displayed in genome browsers, simplifying data aggregation and the integration with other sources of data such as environmental data or downstream analyses. Data conversion also allows the user to analyze the data using genomic tools such as Bedtools. Here you will find some examples of how to use Pergola to explore longitudinally behavioral data in its whole complexity in our tutorial and Pergola documentation.
