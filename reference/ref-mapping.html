Mapping class¶

class mapping.MappingInfo(path, **kwargs)[source]¶

Class holds a dictionary with the mappings between the pergola ontology and the phenomics fields Mapping can be read both from a tabulated file or the _external_mapping_file_format from the Gene Ontology Consortium

path¶
Name of/path to a configuration file
correspondence¶
Dictionary with keys fields in behavioral file and values correspondence in genomic grammar
Returns:MappingInfo object
write(indent=0)[source]¶

Writes correspondence between the pergola ontology terms and the behavioral data

Parameters:indent (0) – int() set indent to be used when writing

Creates a fasta file of the length of the range of value inside the IntData object that will be use for the mapping the data into it

Create a cytoband-like file¶

mapping.write_cytoband(end, start=0, delta=43200, start_phase='light', mode='w', path_w=None, lab_bed=True, track_line=True)[source]¶

Creates a cytoband-like and a bed file with phases of the experiment

Parameters:
  • end – int() last timepoint in the series
  • start – int() first timepoint in the series, by default 0
  • delta – int() delta between intervals, by default 43200 seconds, 12 hours
  • start_phase (light) – str() first phase “light” or “dark”, by default light
  • mode (w) – str() mode to use for file, by default write
  • path_w (None) – str() path to dump the files, by default None
  • lab_bed (True) – If true shows label corresponding to dataType in bed file otherwise shows “.”
  • track_line (True) – If true includes track_line in the file
