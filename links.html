Useful links¶

In this section we gathered all the links that are related to pergola. Our GitHub and DockerHub repositories, webserver and any other information we might find relevant for the user

Web server¶

On our web server you can use main pergola functionalities in a user-friendly interface.


GitHub¶

  • Main GitHub repository where you can find latest Pergola code and you can also clone the repository if you want to contribute to the project.
  • shiny-pergola repository, a Shiny app for the visualization of longitudinal behavioral data processed using pergola
  • Repository wrapping the analysis performed for pergola publication (submitted).
  • Code to reproduce C. elegans analysis of pergola paper.
  • Pipeline to reproduce D. melanogaster analysis of pergola paper.
  • Repository to reproduce M. musculus analysis of pergola paper.

Docker Hub¶

  • Docker image with last pergola release can be found in our Docker Hub repository.
  • Docker image with last shiny-pergola release can be found in our Docker Hub here.
